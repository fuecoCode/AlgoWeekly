## Bitwise操作

### shift ">>" "<<"
x >> 1 右移1位
x << 1 左移1位

e.g., 
x = 21 =>  21=10_1, 10=5_0, 5=2_1, 2=1_0, 1=0_1 > 10101
binary = 0b00010101
hex = 0x15

x >> 1 : 0b00001010 / 10 / 0x0A
x << 1 : 0b00101010 / 42 / 0x2A


## mod2計算

mod2 加法 = XOR

21(00010101) ^ 42(00101010) = 63(00111111)
